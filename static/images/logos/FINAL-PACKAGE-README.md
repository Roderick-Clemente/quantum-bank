## 💡 Design Details

### The Concept
**|0⟩ + |1⟩ = Q**

- **0** (circle) = Traditional banking
- **1** (angled line) = Digital innovation
- **Q** (the fusion) = Quantum Bank
- **Particles** = Quantum uncertainty & cutting-edge tech

### Why the 8° Tilt?
- Makes the "1" flow naturally as the Q-tail
- Still clearly reads as the digit "1"
- Creates visual balance with the circle
- Feels intentional, not broken

### Symmetrical Serifs
- Both top and bottom are 32px wide
- Perfectly balanced visual weight
- Professional and polished
- Works at any scale

## 🖼️ Generate PNGs

**Node.js:**
```bash
npm install sharp
node convert-to-png.js
```

**Python:**
```bash
pip install cairosvg
python3 convert-to-png.py
```

This creates PNGs at: 1024, 512, 256, 128, 180, 32, 16 pixels
