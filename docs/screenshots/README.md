# Screenshots

The main `README.md` references images from this directory. They are not yet
captured — add them here and the README will render them automatically.

## To capture

Run the app locally (`python app.py`, http://localhost:5001) and log in as the
seeded `demo` user, then capture:

| File | Page | How to get there |
|------|------|------------------|
| `dashboard.png` | Account overview | Log in as `demo` → `/dashboard` |
| `login.png`     | Sign-in screen   | `/login` |
| `transfer.png`  | Money transfer   | Log in → `/transfer` |

Suggested width: ~1280px, light theme, window chrome cropped out.
PNG preferred. Keep file names exactly as above so the README links resolve.
